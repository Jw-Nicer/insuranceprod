# **App Name**: Insurance Assistant

## Core Features:

- CSV Upload: Upload CSV files containing insurance claim data.
- Data Visualization: Display charts and tables summarizing the uploaded insurance claim data.
- Interactive Dashboard: Provide a dashboard view with key metrics, charts and filters
- AI Claims Analysis: Use a claims analysis tool powered by generative AI to identify trends, anomalies, and potential areas of concern or opportunity within the uploaded data. This AI-powered tool uses the information present in the CSV.
- Instructions and Guidance: Provide clear instructions and guidance for users on how to use the data uploader effectively.

## Style Guidelines:

- Primary color: A saturated blue (#2979FF) for a sense of trust and security.
- Background color: A very light blue (#F0F8FF) to provide a clean, calming backdrop.
- Accent color: A vibrant purple (#9F00FF) to highlight key elements and calls to action.
- Body and headline font: 'Inter', sans-serif, for a clean, modern, and readable experience.
- Use consistent, clear icons from Lucide to represent different data types and actions.
- Use a clear and intuitive layout to guide users through the data analysis process.
- Use subtle animations for loading states and transitions.
